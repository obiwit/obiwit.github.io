Types of Learning

By Beatriz Borges R, edited on Mar 16, 12018

Machine Learning algorithms can be broadly classified into four separate categories: Supervised Learning, Unsupervised Learning, Semi-Supervised Learning and Reinforcement Learning. They vary in what information they receive – and this in turn affects how they can manipulate the data, and what models they can generate for it.

Supervised Learning

Our data has input and the correct output. With this data, the computer will try to mimic/predict the input data.

Some examples are:

  • Predicting the price of a given product,
  • Filtering spam emails,
  • Recognizing handwritten digits,
  • Captioning an image,
  • Translating between languages.

Unsupervised Learning

Our data has only input. With this data, the computer will try to find structure and relationships in the data.

Some examples are:

  • Clustering data (i.e. market segmentation),
  • Dimensionality reduction.

As an interesting aside, Generative Adversarial Networks (GANs) are technically Unsupervised Learning algorithms that try to mimic the given data.2

Semi-Supervised Learning

Our data has both only input and input with the correct output. Like Supervised Learning, with this data, the computer will try to mimic/predict the input data.

Semi-Supervised Learning is basically a combination of Supervised and Unsupervised Learning, in the sense that you are building a 'Supervised' model while making use of unlabeled data - usually a small fraction of your data will be labelled (the type of input Supervised Learning uses), and the majority will be unlabeled1 (the type of input Unsupervised Learning uses).

An example is:

  • Anomaly detection.

Reinforcement Learning

Our data has input, and an output for a sequence of states. With this data, the computer will try to learn the right sequence of actions that leads to a successful output/conclusion.

Reinforcement learning has a temporal component (in the sense that it learns an action sequence), as opposed to Supervised and Unsupervised Learning.

Some examples are:

  • Playing a game,
  • Making a robot grab a cup.
