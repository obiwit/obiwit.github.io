Types of Learning

Machine Learning Algorithms can be broadly classified into three separate categories: Supervised Learning, Unsupervised Learning and Reinforcement Learning. They vary in what information they receive – and this in turn affects how they can manipulate the data, and what models they can generate for it.

Supervised Learning

We have input and the correct output. Tries to mimic/predict the input data.

Some examples are:

  • Predicting house prices,
  • Filtering spam emails,
  • Recognizing handwritten digits,
  • Captioning an image,
  • Translating between languages

Unsupervised Learning

We have only input. Tries to find structure and relationships from the data.

Some examples are:

  • Clustering data,
  • Dimensionality reduction

Reinforcement Learning

We have input, and an output for a sequence of states. Tries to learn the right sequence of actions that leads to a successful output/conclusion.

Reinforcement learning has a temporal component (in the sense that it learns an action sequence), as opposed to Supervised and Unsupervised Learning.

Some examples are:

  • Playing a game,
  • Making a robot grab a cup
