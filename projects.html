ExpROVer

An open source solution supporting the 'VideoRay Pro 4' ROV

The ExpROVer is an open source solution enabling the remote control of Remotely Operated Vehicles (ROVs), through an easy and intuitive user interface (via dedicated Web and Android applications); low-level communication (using Linux), and integration with ROS (the Robot Operating System), and ROSbridge-powered server capabilities; several extensions, like Computer Vision capabilities (fish detection using a modified YOLOv3 architecture), and a set of semi-autonomous maneuvers.
