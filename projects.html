Smart Library

An automatic and semi-autonomous self-organizing text library

A system which could automatically detect a text's language, and sort it into one of several categories according to its subject. MinHashes, a vanilla bloom filter (used to check whether the file already existed in the library), and a counting bloom filter (designed to supplement the 'choice' of a given text's subject) were used.

100 Cerebros Image Gallery

An online application allowing you to upload photos, to which you can add several effects, or turn into a meme. You can preview and undo each one of the effects (and meme captions).
